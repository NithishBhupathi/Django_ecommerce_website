Django E-Commerce Website

This is a full-stack e-commerce web application built using Django.
It allows users to browse clothing products, filter by category, add items to cart, manage quantities, and place orders through a checkout system.

The project demonstrates core e-commerce functionalities with a clean UI and user authentication.

🚀 Features
-----------------------

👕 Product listing (Men, Women, Kids)

🔍 Category-based filtering (Fashion dropdown)

🛒 Add to Cart & Quantity Management

🔐 User Authentication (Login / Logout)

📦 Checkout with Address & Phone

🎨 Responsive UI with CSS

🧾 Order creation & cart clearing

-------------------------------------------

🛠️ Technologies Used
------------------------
Backend: Django (Python)

Frontend: HTML, CSS, JavaScript

Database: SQLite (default Django DB)

Authentication: Django Auth System

Version Control: Git&GitHub

--------------------------------------------------

⚙️ Installation & Setup
---------------------------------------
1. Create Virtual Environment
   
          python -m venv env

   Activate:
 
        env\Scripts\activate

3. Clone the Repository
    
    git clone
   
    cd to project folder
   
5. Install Required Libraries
   
   pip install django
   
   pip install pillow

7. Apply Migrations
   
   python manage.py makemigrations
   
   python manage.py migrate

9. Create Superuser (Admin)
    
   python manage.py createsuperuser

11. Run the Server
    
    python manage.py runserver
